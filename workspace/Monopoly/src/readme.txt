1] Eclipse
	load project in eclipse and run the TestMonopoly.java file
	- you can give argument as "-dice" to "Run configuration" option
	- if you dont give argument, system will throw the dice

2] Command line
	go to folder containing TestMonopoly.java file
	- run "javac TestMonopoly"
	- run "java TestMonopoly"   for machine run
	- run "java TestMonopoly -dice"  for user to throw dice
